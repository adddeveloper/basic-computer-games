=== 6502 WindowBlind (New) Cursor Set ===

By: codewar65

Download: http://www.rw-designer.com/cursor-set/6502-windowblind-new

Author's description:

A retro cursor set using the Commodore 64 16 color palette. 3D rendered in POV-Ray; post processed in Corel Photo-Paint, IC Warlock; assembled in RealWorld Cursor Editor. 

Created to accompany my 6502 WindowBlind (for use with Stardocks WindowBlind 10).
